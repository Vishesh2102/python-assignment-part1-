# Student Grade Tracker - Python Basics
This project demonstrates fundamental Python concepts including control flow, string manipulation, and basic data types.

### Features:
- **Data Cleaning:** Uses `.strip()` and `.title()` to normalize messy user input.
- **Grade Logic:** Implements an `if-elif-else` chain to categorize student performance.
- **Input Handling:** A `while` loop allows dynamic entry of new subjects with `isdigit()` validation to prevent crashes.
- **Reporting:** Uses f-string formatting to create a clean, readable performance summary.
